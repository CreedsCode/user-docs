# What is TalentLayer Indie?

{% hint style="info" %}
**Good to know:** TalentLayer Indie is the first platform built on TalentLayer Protocol.&#x20;
{% endhint %}

TalentLayer Indie is a reputation tool for independent freelancers. Similar to how you can build up a reputation over time by completing work and receiving reviews on freelancing platforms like Upwork or [Freelancer.com](http://freelancer.com), now you and your independent clients can create verified reviews to signal one anotherâ€™s trustworthiness.

TalentLayer Indie is the first implementation of the TalentLayer Protocol, a decentralized backend for talent marketplaces and sovereign work reputation system.
